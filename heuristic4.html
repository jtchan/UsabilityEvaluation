Consistency and Standards

"Users should not have to wonder whether different words, situations, or actions mean the same thing. Follow platform conventions." - Jakob Nielsen

A website should not drastically change between related pages. Maintaining consistency between pages is important for keeping the user from becoming confused or distracted during interaction.

Consistent Layout

As far as general layout goes, GSU.edu does a good job keeping everything consistent and familiar. A title/header is always present on the top left, and a general navigation bar is always shown at the top of the page, in the same style with hover menus and red underlines under the currently shown page.

Unfamiliar Sub-Sites

There is an inconsistency problem when it comes to the pages that are linked from the homepage, which can be accessed from a dropdown menu in the navigation bar on the frontpage, as well as the universal drop down menu on each of the site's subpages. These pages look nothing like the homepage and, in fact, look like completely different websites unaffiliated with the university. While this may be understandable because these sites are for varying sections of the university, such as admissions or sports or research, there are so many sections that the different color header and color schemes are overwhelming.

Home Page Button

For some of the pages that are directly linked from the second navigation bar on the homepage, there is an icon of a logo of the university, which the user can click to head back to the homepage. However, this is only true for the first two tabs of the navigation bar. All the other pages, with the exception of the “Alumni” page, have the icon as well, but clicking on it only refreshes the current page the user is on. The “Alumni” page has a logo as well, but its color scheme and design is completely different from the logos used on the other pages. There is not a consistent method for the user to return to the homepage using these logos.
