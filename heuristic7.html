Flexibility and Efficiency of Use

"Accelerators -- unseen by the novice user -- may often speed up the interaction for the expert user such that the system can cater to both inexperienced and experienced users. Allow users to tailor frequent actions." - Jakob Nielsen

If a user is new to the page, it's good to have clearly structured organization. However, if it takes ten clicks to get the to page they want, the more advanced user will get frustrated.

Can't get back to the homepage

On the homepage, there is a navigation bar that includes "Athletics" as a category. This is a link to "georgiastatesports.com". Once on this page, there is no option on the navigation bar to go home to "gsu.edu". This is very unflexible for both the novice user and advanced. Also, it is inefficient because instead of having an easily clickable link, the user either has to type in the web address again or search through multiple pages to go home.

Doesn't allow for subnavigation

The user has to mouse through the first dropdown to get to a page. However, what if the user wants to easily get to a page from the subnav? There is no option in the dropdown to have another dropdown. The more advance user, and also the novice user has to go to the first destination and then navigate from there. Also, most of the pages on the second level don't have very functional or flexible navigation bars.
